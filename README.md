# Superstore Sales Dashboard

## Project Overview
Interactive Excel Dashboard built on US Retail Sales data.

## Dataset
- Source: Kaggle (Superstore Dataset)
- Rows: 9,994
- Period: 2014-2017

## Features
- Dynamic Slicers (Year, Region)
- Timeline Filter
- Dynamic KPI (Total Sales, Profit, Orders)
- 4 Interactive Charts

## Charts
- Monthly Sales Trend (Line Chart)
- Region Wise Sales (Bar Chart)
- Category Wise Profit (Pie Chart)
- Top 10 Products (Bar Chart)

## Key Findings
- West region has highest sales ($725,457)
- Sales grew 50% from 2014 to 2017
- Technology category most profitable

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers & Timeline
